# TinyMind_OCR
参加Tinymind书法识别大赛的源码
***
## 本代码参考[TinyMind-start-with-0](https://github.com/Link2Link/TinyMind-start-with-0)    
***
## 结构    
本代码在参考基础上，修改了网络结构，并增加了VGG16网络的调用    
## 结果    
在只有CPU的基础上，由于时间原因，只训练了两个epoch，取得了82.79%的成绩，如果继续优化，结果应该会更好
